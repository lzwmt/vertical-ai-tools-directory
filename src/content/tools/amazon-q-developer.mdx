## 结论

Amazon Q Developer 适合“代码问题和云问题经常连在一起”的团队，而不只是一般意义上的代码补全工具。

## 场景适配

如果团队经常在写业务代码的同时处理 AWS 架构、权限、部署、升级和故障问题，Amazon Q Developer 的上下文优势会更明显。它不是最通用的默认项，但对 AWS 团队来说很有针对性。

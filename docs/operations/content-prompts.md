# 内容生产 Prompt 模板包

## 使用原则
- 让 AI 先做结构化整理，再做正文生成
- 永远要求 AI 标记“不确定信息”
- 任何涉及结论判断的部分都需要人工复核

## 1. 工具信息整理 Prompt

```text
你现在是一个 AI 工具导航站的内容研究员。

请根据我提供的工具官网、定价页、文档页信息，整理成结构化字段。

输出要求：
1. 只输出事实，不要编造
2. 如果某项信息无法确认，写“待确认”
3. 输出为 Markdown

字段：
- 工具名
- 官网
- 一句话描述
- 主要分类
- 适用场景
- 定价模式
- 是否支持中文
- 是否提供 API
- 是否适合团队
- 核心功能
- 潜在替代品
- 需要人工验证的问题

输入资料：
[粘贴官网 / 定价页 / 文档摘要]
```

## 2. 工具页初稿 Prompt

```text
你现在是一个中文 AI 工具导航站的编辑助手。

请根据以下结构化资料，生成一篇工具详情页初稿。

要求：
1. 面向中文内容创作者和运营人员
2. 不要照抄官网宣传语
3. 语言要直接、实用
4. 明确写出适合谁、不适合谁
5. 如果某个点无法确认，要写“待人工验证”

文章结构：
- 工具概述
- 适合什么场景
- 核心功能
- 优点
- 缺点
- 更适合谁
- 不适合谁
- 和替代品相比
- 简短结论

输入资料：
[粘贴结构化字段]
```

## 3. 对比页生成 Prompt

```text
你现在是一个 AI 工具导航站的对比页编辑。

请比较这两个工具，并生成中文对比页初稿。

要求：
1. 不要只列功能，要给出选型建议
2. 面向中文内容创作者和运营人员
3. 强调中文支持、价格、上手门槛、中国区可访问性
4. 结论部分必须回答“什么人该选 A，什么人该选 B”

输出结构：
- 对比概述
- 适合人群差异
- 价格差异
- 中文支持差异
- 中国区使用差异
- 功能侧重点差异
- 推荐结论

工具 A：
[资料]

工具 B：
[资料]
```

## 4. 场景页生成 Prompt

```text
你现在是一个 AI 工具导航站的场景页编辑。

请围绕“[具体场景]”生成一篇中文场景推荐页初稿。

要求：
1. 面向中文内容创作者和运营人员
2. 不做空泛推荐，要说明每个工具为什么适合这个场景
3. 每个工具都写一句明确推荐理由
4. 最后给出“新手先选谁、进阶用户选谁、团队选谁”

输出结构：
- 场景说明
- 推荐标准
- 工具推荐列表
- 每个工具的推荐理由
- 适合不同用户的选择建议
- 总结

候选工具：
[工具列表和资料]
```

## 5. SEO 标题与描述 Prompt

```text
请为下面这篇页面内容生成：
- 5 个 SEO 标题候选
- 3 个 meta description 候选

要求：
1. 中文
2. 不夸张，不标题党
3. 适合内容型 AI 工具导航站
4. 优先高意图搜索表达

页面类型：
[工具页 / 对比页 / 场景页]

页面主题：
[主题]

页面摘要：
[摘要]
```

## 6. 社媒摘要 Prompt

```text
请把下面这篇内容改写成适合社媒分发的短内容。

输出：
- 1 条小红书风格短文案
- 1 条公众号摘要
- 1 条即刻 / 社群摘要

要求：
1. 重点突出结论
2. 不要过度营销
3. 保留工具名和适用场景

原文摘要：
[摘要]
```

## 7. 周运营复盘 Prompt

```text
你现在是这个 AI 工具导航站的运营分析助手。

请根据以下周数据做复盘，并输出下周建议。

请重点分析：
- 哪些页面有流量
- 哪些页面有点击
- 哪些页面有流量但点击弱
- 哪些页面应该更新
- 下周最值得新增的 5 个选题

输出结构：
- 本周发现
- 高价值页面
- 问题页面
- 更新建议
- 下周选题建议

数据：
[粘贴页面、PV、点击、订阅、提交等数据]
```
